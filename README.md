# 🦷 AI Dental Receptionist (Voice Agent)

An AI-powered voice receptionist that answers patient calls, handles queries, and books appointments using natural conversation.

## ✨ Features

* 📞 Simulates inbound phone calls
* 🧠 Understands natural language
* 📅 Books appointments step-by-step
* 👤 Collects patient details
* ❓ Answers FAQs (hours, services, location)
* 🎙️ Human-like voice interaction

---

## 🛠 Tech Stack

* Vapi (Voice AI)
* OpenAI (LLM)
* ElevenLabs (Text-to-Speech)
* Deepgram (Speech-to-Text)

---

## 🧠 How It Works

1. User clicks “Start Call”
2. AI starts voice conversation
3. Identifies intent (booking / question)
4. Collects required details
5. Confirms and books appointment

---

## 📂 Project Structure

* `index.html` → Web demo interface
* `system_prompt.txt` → AI behavior
* `tools.json` → Function calling setup

---

## 📞 Call Simulation

This project uses Vapi’s web SDK to simulate real inbound calls directly in the browser.

No external telephony setup (like Twilio) is required.
